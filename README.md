# Space Suit Thermal and Radiation Analysis (LEO)

A transient 1D numerical simulation of heat transfer and radiation attenuation for a multilayer space suit under Low Earth Orbit (LEO) conditions.

This repository was developed to accompany an academic paper presentation. It combines a spatially-dependent implicit finite difference thermal model (Crank-Nicolson) with an empirical mass-attenuation radiation proxy to evaluate the survivability of common space suit materials (Kevlar, Nomex, and MLI).

## 🚀 Overview

Space suits must maintain strict thermal boundaries while providing structural and radiation protection in extreme orbital environments. This simulation models:

* Transient heat conduction through dissimilar material layers.
* Explicit coupling of orbital environmental fluxes (Solar + Earth IR).
* Radiative heat exchange at the deep-space boundary.
* Empirical mass-attenuation shielding against X-ray, ~1 MeV, and GCR proxies.

## 🧮 Mathematical & Numerical Framework

This model goes beyond basic constant-property simulations by implementing rigorous numerical methods:

### Thermal Conduction Solver

The system solves the 1D heat equation with spatially varying thermal properties:


$$\rho C_p \frac{\partial T}{\partial t} = \frac{\partial}{\partial x} \left( k(x) \frac{\partial T}{\partial x} \right)$$

* **Integration Scheme:** Crank-Nicolson (unconditionally stable, 2nd-order accuracy).
* **Material Interfaces:** Handled using the **Harmonic Mean** of thermal conductivities to ensure strict heat flux continuity between dissimilar layers.
* **Boundary Physics:** The outer net-flux boundary is governed by:

$$q_{net} = \alpha_s q_{solar} + \varepsilon q_{IR} - \varepsilon\sigma(T_{outer}^4 - T_{space}^4)$$



This is strictly coupled to the numerical matrix using a **Ghost-Node formulation** to prevent artificial energy creation/loss at the surface node.

### Radiation Attenuation Proxy

Evaluates primary particle flux mitigation using a mass-attenuation equivalent model ($\text{cm}^2/\text{g}$) to account for varying material densities across the stack:


$$I = I_0 \exp(-\mu \cdot \rho_{areal})$$

## 🧵 Material Stack Configuration

| Layer | Material | Thickness | Function |
| --- | --- | --- | --- |
| **Outer** | Kevlar | 3 mm | Structural integrity, micrometeoroid protection |
| **Middle** | Nomex | 2 mm | Thermal resistance, fire retardance |
| **Inner** | MLI | 10 mm | Primary vacuum thermal barrier ($k_{eff}$) |

## ⚙️ Installation & Usage

**Dependencies:**
The script relies entirely on standard Python scientific computing libraries. No specialized FEM software is required.

```bash
pip install numpy matplotlib

```

**Execution:**
Run the primary script to simulate the normal LEO case (2 orbits) and generate the analysis dashboard.

```bash
python space_suit_thermal_analysis.py

```

*(To test edge cases like "Extreme Hot" or "Extreme Cold", change the `CASE` variable at the top of the script).*

## 📊 Outputs & Visualizations

Upon execution, the script prints a comprehensive engineering summary to the console and generates a high-resolution dashboard (`space_suit_thermal_analysis.png`) containing:

1. **Temperature Profiles:** Spatial distribution of heat through the layers over time.
2. **Surface Temperature Evolution:** Outer and inner boundary responses across multiple 90-minute orbits.
3. **Orbital Heat Flux:** Net external heating profile transitioning between sunlight and eclipse phases.
4. **Radiation Attenuation Benchmarks:** Comparative shielding effectiveness of different material permutations.
5. **Material Property Matrix:** Visual comparison of thermal conductivity ($k$) and diffusivity ($\alpha$).

## ⚠️ Assumptions & Limitations

* **1D Heat Flow:** Assumes heat flows purely perpendicular to the suit surface (no lateral conduction).
* **Equivalent Bulk MLI:** Multi-Layer Insulation is modeled using an effective bulk thermal conductivity ($k_{eff}$) rather than an explicit multi-node radiation network.
* **No Spallation:** The radiation model is an empirical proxy for primary particle attenuation. It does *not* account for secondary radiation fragments (spallation) generated by high-energy Galactic Cosmic Rays (GCRs) interacting with the shielding.
* **Vacuum Environment:** Internal convection and suit life-support fluid loops are not coupled to the inner boundary condition.
